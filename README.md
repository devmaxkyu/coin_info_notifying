# Crypto Currency PHP / Mailer Script
A php script to make automation to get and parse email, send return email with result from api

## Getting Started
Simply clone this repository to your local machine using 'git clone'

### Prerequisites

The PHP IMAP Extension are necessary to enable in your php.ini file

### Running Script

Prior to run this script, You will need set your email configration

## Running the tests

No tests
